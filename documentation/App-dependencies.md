# App Dependencies

The dependencies required to run the project :

- `Node` v16.14.2 

- `npm` 6.14.8 (LTS) or more recent

- `AWS CLI` v2

- `RDS` Postgres Database.

- `S3` bucket for hosting uploaded files.

- `Angular` 8.2.14 for the frontend.

- `bcrypt` 5.0.1

- `body-parser` 1.18.3

- `dotenv` 8.2.0

- `express` 4.16.4

- `jsonwebtoken` 8.5.1

- `pg` 8.7.1

- `sequelize` 5.21.4
